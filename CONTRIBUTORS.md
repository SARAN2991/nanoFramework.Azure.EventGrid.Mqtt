## How to Contribute

We welcome contributions! Here's how you can help:

1. **Report issues** — File bugs or feature requests via [GitHub Issues](https://github.com/SARAN2991/nanoFramework.Azure.EventGrid.Mqtt/issues)
2. **Submit pull requests** — Fork, branch, and submit a PR against `main`
3. **Improve documentation** — Fix typos, clarify guides, add examples
4. **Test on hardware** — Report results from ESP32 or other nanoFramework boards

### Contribution Guidelines

- Follow existing code style and naming conventions
- Add XML doc comments (`/// <summary>`) to all public APIs
- Ensure the project builds cleanly before submitting
- Keep commits atomic and write clear commit messages
- For new features, include a sample in the `samples/` directory

### Code of Conduct

This project follows the [.NET Foundation Code of Conduct](https://dotnetfoundation.org/code-of-conduct).
All participants are expected to uphold this code.
